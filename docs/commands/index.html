    • Triggers

    Triggers is the actual command. Which actually goes into million parts, but we need only 2 part to use it. Or sometime 3.

    Part 1: Prefix

    Part 2: Command

    Part 3: Option (as Suffix)

    So if the prefix is "/" alias Slash-symbol, and the command is "help", it would be "/help"





    Conditions

    Conditions is something like "If" the user has a role, or doesn't have a role. Or if their User ID is same than the value marked up, you can't run the command succesfully.
